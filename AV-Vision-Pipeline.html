🎯 Project Overview

This project demonstrates a complete web-based object detection system for autonomous vehicle applications. It combines computer vision, web APIs, and modern Python development practices to create a working system that can detect vehicles, pedestrians, and traffic infrastructure in real-time.

🔍 Object Detection

YOLO-based detection of vehicles, pedestrians, and traffic infrastructure with confidence scoring and bounding box visualization.

🌐 Web API

RESTful FastAPI backend with automatic documentation, error handling, and file upload capabilities.

📱 Interactive Frontend

Streamlit web interface for easy image upload, real-time detection, and results visualization.

🧪 Testing & Quality

Unit tests with pytest, proper error handling, and professional code structure with virtual environments.
